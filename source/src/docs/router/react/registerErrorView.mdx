# registerErrorView

Registers a default error view to be displayed when view component failed to render.
The previous registered error view, if any, will be overriden when called multiple times.

See [Error view](/docs/router/react/error-view) for usage.

## Syntax

```javascript
registerErrorView(component)
```

### Parameters

<dl>
    #### `component`
    A react component that receives [`ErrorViewProps`](:).

</dl>
