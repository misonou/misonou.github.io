# isViewMatched

Determines whether a registered view component matches current route state.
However it does not imply if the view is in fact being rendered.

> Use [`isViewRendered`](:) when possible and appropriate.

## Syntax

```javascript
isViewMatched(view)
```

### Parameters

<dl>
    #### `view`
    A view component returned from [`registerView`](:).

</dl>
