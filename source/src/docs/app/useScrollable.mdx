# Scrollable extension

Enables auto scrollable area initialization and scrolling events.
Requires the [scrollable plugin](https://github.com/misonou/jquery-scrollable).

> See [Enabling extension](/docs/app/create-app#s-enabling-extensions) on how to use
  extension for brew.js app.

<ImportHint
    name="Scrollable"
    module="brew-js/extension/scrollable"
    import="default"
    expanded
/>

## Syntax

```javascript
app.useScrollable()
app.useScrollable(defaultOptions)
```

<dl>
    #### `defaultOptions` <Badge.Optional />
