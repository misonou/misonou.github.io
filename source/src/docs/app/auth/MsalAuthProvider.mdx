## Static methods

### `MsalAuthProvider.create()`

```javascript
create(key, clientId, authority)
create(key, clientId, authority, scopes)
create(key, client)
create(key, client, options)
create(key, options)
```

Creates an authentication provider using default configurations.

### Parameters

<dl>
    #### `key`
    A unique key.

    #### `clientId`
    Client ID of the app registered in Azure's application registration portal.

    #### `client`
    An MSAL client application.

    #### `authority`
    Authority URI, typically `https://login.microsoftonline.com/common` or one associated with the tenant.

    #### `options` <Badge.Optional />
    A dictionary containing options.

    #### `scopes` <Badge.Optional />
    Array of scopes the application is requesting access to.

</dl>

### Return value

An authentication provider which can be passed to [`app.useAuth`](:).

### `MsalAuthProvider.setDefault()`

```javascript
setDefault(config)
```

Sets default configurations for MSAL client application, which will be applied when
an authentication provider is created without explicitly supplying a [`PublicClientApplication`](:) instance.

> Note that it must be called before creating provider.

### Parameters

<dl>
    #### `config`
    A subset of configurations which is merged into [`MsalAuthProvider.defaultConfig`](:).

</dl>
