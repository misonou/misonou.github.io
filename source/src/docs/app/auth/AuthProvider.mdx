## Static methods

### `AuthProvider.from()`

```javascript
from(key, client)
```

Creates an authentication provider from a stateless authentication client.

### Parameters

<dl>
    #### `key`
    A unique key identifying the provider.

    #### `client`
    A client that implements authentication request to server. The client should implements the [`AuthClient`](:) interface.

</dl>

### Return value

An authentication provider which can be passed to [`app.useAuth`](:).
