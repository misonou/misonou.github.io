# Maths functions

### `max`, `min`

Clamps the input value at the given maximum or minimum value.

<WaterpipeExample
    data={{ value: 100 }}
    examples={['value max 10', 'value max 1000', 'value min 10', 'value min 1000']}
/>

### `round`, `floor`, `ceil`

Rounds the input value to the integer.

<WaterpipeExample
    data={{ value: 10.4 }}
    examples={['value round', 'value floor', 'value ceil']}
/>

### `plus (+)`, `minus (-)`, `multiply (*)`, `divide (/)`, `mod (%)`, `pow (^)`

Performs basic arithmetic operations.

<WaterpipeExample
    data={{ value: 10 }}
    examples={['value + 1', 'value - 1', 'value * 2', 'value / 2', 'value % 2', 'value ^ 2']}
/>
