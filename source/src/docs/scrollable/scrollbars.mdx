# Scrollbars

## CSS variables

~~1.11.0~~

Scrollbar can be styled by the following CSS variables:

- `--jqs-scrollbar-inset`
- `--jqs-scrollbar-size`
- `--jqs-scrollbar-color`
- `--jqs-scrollbar-radius`
- `--jqs-scrollbar-opacity`

> The corresponding `scrollbarInset`, `scrollbarSize` and `scrollbarStyle` (`backgroundColor`, `borderRadius`, `opacity`) options,
  when specified, will override values from CSS variables.

<Snippets
    snippets={[
        '--jqs-scrollbar-inset: 0',
        '--jqs-scrollbar-size: 10px',
        '--jqs-scrollbar-color: blue',
        '--jqs-scrollbar-radius: 0',
        '--jqs-scrollbar-opacity: 1',
    ]}
    rootProps={{
        scrollable: 'auto',
        style: { maxHeight: 300, overflow: 'hidden' }
    }}
    render={(index, ref, value) => {
        useLayoutEffect(() => {
            const style = ref.current.style;
            for (let i = style.length - 1; i>=0; i--) {
                if (style[i].startsWith('--jqs')) {
                    style.removeProperty(style[i]);
                }
            }
            const [_, name, val] = /^([^:]+):(.+)/.exec(value);
            style.setProperty(name, val);
        }, [index]);
        return (
            <div scrollable-target="" style={{ maxWidth: 300 }}>
                <p><Lipsum /></p>
                <p><Lipsum /></p>
                <p><Lipsum /></p>
            </div>
        );
    }} />


## CSS class

CSS class names can be added to rendered scrollbars by `scrollbarClass` option.

For example if the option is set to `app-scrollbar`, the horizontal and vertical scrollbars will
have the class names `app-scrollbar app-scrollbar-x` and `app-scrollbar app-scrollbar-y` respectively.

```javascript
$(element).scrollable({
    scrollbarClass: 'app-scrollbar'
    /* ... */
});
```

## Styling options

Scrollable can also be styled by `scrollbarInset`, `scrollbarSize` and `scrollbarStyle` options.

> Styling by CSS variables is recommended. Styling options are only effective
  during setup. Updating by [`Scrollable.setOptions`](:) will not take effect.

```javascript
$(element).scrollable({
    scrollbarInset: 3,
    scrollbarSize: 10,
    scrollbarStyle: {
        opacity: 0.7
    }
    /* ... */
});
```

## Disabling scrollbars

Scrollbar can be disabled by setting `scrollbar` option to `false`.

> The `scrollbar` option is only effective
  during setup. Updating by [`Scrollable.setOptions`](:) will not take effect.

```javascript
$(element).scrollable({
    scrollbar: false
    /* ... */
});
```
