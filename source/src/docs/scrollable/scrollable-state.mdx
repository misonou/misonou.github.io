# Scrollable state

By default the plugin adds several state classes on the container element for fine-tuning styles for different state.

<dl>
    #### `scrollable-x`
    Indicates if the content is scrollable in x-direction.

    #### `scrollable-x-l`
    Indicates if the content is scrollable in x-direction to the left (when [`Scrollable.scrollX`](:) is non-zero).

    #### `scrollable-x-r`
    Indicates if the content is scrollable in x-direction to the right (when [`Scrollable.scrollX`](:) is not at maximum).

    #### `scrollable-y`
    Indicates if the content is scrollable in y-direction.

    #### `scrollable-y-d`
    Indicates if the content is scrollable in y-direction downwards (when [`Scrollable.scrollY`](:) is non-zero).

    #### `scrollable-y-u`
    Indicates if the content is scrollable in y-direction upwards (when [`Scrollable.scrollY`](:) is not at maximum).

    #### `scrolling`
    Whether scrolling is in action.
</dl>

> The name of the CSS classes can be controlled by `scrollableXClass`, `scrollableYClass`, `scrollingClass` options.
