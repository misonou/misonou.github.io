# Leading content

The scrollable plugin will consider any element rendered before the content container to be fixed in position.
Therefore add a fixed header only requires proper CSS styling.

```html
<div scrollable>
    <h4>Header</h4>
    <div scrollable-target>
        <!-- ... -->
    </div>
</div>
```
