# cancelLock

Requests cancellation of all asynchronous operations.

## Syntax

```javascript
cancelLock()
cancelLock(element)
cancelLock(element, force)
cancelLock(element, reason)
```

<ImportHint
    name="cancelLock"
    module="zeta-dom/domLock"
    global="zeta.dom.cancelLock"
/>

### Parameters

<dl>
    #### `element` <Badge.Optional />
    A DOM element.

    #### `force` <Badge.Optional />
    Whether to disregard cancellation handler and forcibly release all locks.

    #### `reason` <Badge.Optional /> <Badge.Version />
    A [`CancellationRequest`](:) object that will be passed to cancellation handling callback.
</dl>

### Return value

A promise which is fulfilled when all operations are cancelled; or is rejected otherwise.
