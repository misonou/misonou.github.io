# unsetTabRoot

Unmarks an element to be a tab root, so that effect by [`setTabRoot`](:) is reverted.

## Syntax

```javascript
unsetTabRoot(element)
```

    name="unsetTabRoot"
    module="zeta-dom/dom"
    global="zeta.dom.unsetTabRoot"
/>

### Parameters

<dl>
    #### `element`
    A DOM element.
