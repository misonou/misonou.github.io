# focus

Sets focus to the given element.

## Syntax

```javascript
focus(element)
focus(element, focusInput)
```

<ImportHint
    name="focus"
    module="zeta-dom/dom"
    global="zeta.dom.focus"
/>


### Parameters

<dl>
    #### `element`
    A DOM element.

    #### `focusInput` <Badge.Optional /> <Badge.Version />
    Whether to focus the first focusable element within. Default is `true`.

</dl>
