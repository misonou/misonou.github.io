# toQueryString

Returns a query string that includes the specific parameters

## Syntax

```javascript
toQueryString(values)
```

### Parameters

<dl>
    #### `values`
    A dictionary containing parameter names and associated values. Values with `false`, `null` or `undefined` are ignored.

</dl>
