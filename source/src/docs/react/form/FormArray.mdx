# FormArray component

See [Data object and array](/docs/react/form/data-object-array) for usage.

## Syntax

```tsx
<FormArray name={name} {...optionalProps}>
    {children}
</FormArray>

<FormArray value={value} {...optionalProps}>
    {children}
</FormArray>
```

### Properties

<dl>
    #### `name`
    Specifies the name of property of which the data object is accessible in parent data object.

    #### `value`
    Specifies the data item contained in parent data array.

    #### `children`
    React node, or a callback that returns react node.
