# ToggleField class

The `ToggleField` class are not supposed to be instantiate manually. Instead it is used to
augment extra functionalities when used with [`useFormField`](:).

<ImportHint
    name="ToggleField"
    module="zeta-dom-react"

    expanded />

## ToggleFieldProps interface

<MemberList

### `ToggleFieldProps.checked`

Whether the field is checked.

### `ToggleFieldState.toggleValue()` <Badge.Version />
