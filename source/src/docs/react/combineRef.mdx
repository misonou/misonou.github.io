# combineRef

Combines multiple mutable ref objects or ref callbacks into a single ref callback.

## Syntax

```javascript
combineRef()
combineRef(ref0)
combineRef(ref0, ref1)
combineRef(ref0, ref1, /* … ,*/ refN)
```

### Parameters

<dl>
    #### `ref0` … `refN` <Badge.Optional />
    List of refs.

</dl>
