# useEagerState

Similar to but unlike `React.useState`, it guarantees that callback supplied to updating function is called synchronously,
and it will not cause component to re-render when state did not change.

## Syntax

```javascript
useEagerState()
useEagerState(initialState)
```

<ImportHint
    name="useEagerState"

### Parameters

<dl>
