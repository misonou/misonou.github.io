# createErrorHandler

Creates an [`ErrorHandler`](:) object that allows capturing errors from descendant elements
or emitting errors to parent elements of a specified container element.

> See [`ErrorHandler`](:) for more examples.

## Syntax

```javascript
createErrorHandler(element)
```

<ImportHint
    name="createErrorHandler"
    module="zeta-dom-react"

    global="zeta.react.createErrorHandler" />


### Parameters

<dl>
    #### `element`
    A DOM element.

</dl>

### Return value

An [`ErrorHandler`](:) object.
