# keys

Returns all keys for the given prefix.

> This function is returned from [`makeTranslation`](:).

## Syntex

```javascript
keys(prefix)
```

## Example

```typescript
const resources = {
    en: {
        prefix: {
            key1: 'String 1'
            key2: 'String 2'
        }
    },
    /* ... */
};

const { keys } = makeTranslation(resources, 'en');

keys('prefix') // -> ["key1", "key2"]
```
