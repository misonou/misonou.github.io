# Using I18n in React

The `makeTranslation` function is provided in conjunction to the i18n extension in brew.js.

> The app must setup i18n in order to work, see [`app.useI18n`](:) for instructions.

## Creating translation from JSON

```typescript
const resources = {
    // first level are languages
    en: {
        // second level are prefixes
        prefix: { key: 'Translation string' }
    },
    de: {
        prefix: { key: 'Ãœbersetzungszeichenfolge' }
    }
};

const { translate, useTranslation } = makeTranslation(resources, 'en');
```

Now it is ready to [get the translations](/docs/react/i18n/getting-translation).
