# getTranslation

Returns a translate callback.

> This function is returned from [`makeTranslation`](:).

## Syntax

```javascript
getTranslation()
getTranslation(prefix0)
getTranslation(prefix0, prefix1)
getTranslation(prefix0, prefix1, /* … ,*/ prefixN)
```

### Parameters

<dl>
    #### `prefix0` … `prefixN` <Badge.Optional />
    Create translation callback which only looks up the given prefixes.

    If no prefixes is given, prefix must be included in the key
    supplied to translate callback.
</dl>

### Return value

A [`translate`](:) callback.
