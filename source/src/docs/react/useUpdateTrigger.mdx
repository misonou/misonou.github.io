# useUpdateTrigger

Creates a callback that forcibly triggers re-render of a component.

## Syntax

```javascript
useUpdateTrigger()
```

<ImportHint
    name="useUpdateTrigger"
    module="zeta-dom-react"
    require="useUpdateTrigger"
    global="zeta.react.useUpdateTrigger" />

### Return value

A callback that forcibly triggers re-render of a component.
