# useUnloadEffect

Registers a cleanup callback to be invoked when the component is being unmounted,
or the document is being unloaded.

## Syntax

```javascript
useUnloadEffect(callback)
```

<ImportHint
    name="useUnloadEffect"
    module="zeta-dom-react"
    require="useUnloadEffect"
    global="zeta.react.useUnloadEffect" />

### Parameters

<dl>
    #### `callback`
    Callback to be invoked, receiving a boolean indicating if the page is still cached.
    The flag is always `false` if the component is being unmounted.

    ```tsx
    function Component() {
        useUnloadEffect(persisted => {
            if (persisted) {
                // user is leaving the page and
                // the page is cached in backword/forward cache
            }
        });
    }
    ```
</dl>
