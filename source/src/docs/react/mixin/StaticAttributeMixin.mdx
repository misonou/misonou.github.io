## Constructor

```javascript
new StaticAttributeMixin(name, value)
new StaticAttributeMixin(attributes)
```

### Parameters

<dl>
    #### `name`
    Name of attribute.

    #### `attributes`
    A dictionary containing attributes and their values to be set on rendered elements.

    #### `value`
    Value to be set. Default is empty string.
