# ClassNameMixin class

Base class for implementing mixins that requires certain CSS class names to be
preserved after component update.

<MemberList
    i="ClassNameMixin"
    extends={['Mixin', 'StatefulMixin']}
    extendsTo={['AnimateMixin', 'FlyoutMixin', 'FlyoutToggleMixin', 'FocusStateMixin', 'LoadingStateMixin', 'ScrollableMixin', 'UnmanagedClassNameMixin']}
    im={['onClassNameUpdated()']}
/>

## Constructor

```javascript
new ClassNameMixin(classNames)
```

### Parameters

<dl>
    #### `classNames`
    Specifies a list of class names to be obeserved.

</dl>

## Instance methods

### `ClassNameMixin.onClassNameUpdated()`

```javascript
onClassNameUpdated(element, prevState, state)
```

Override to perform actions when observed class names has changed.

### Parameters

<dl>
    #### `element`
    DOM element which its class names has changed.

    #### `prevState`
    Previous states representing the presence of each class name.

    #### `state`
    Current states representing the presence of each class name.

</dl>
